Detailed Description

template<typename ModelType>
class ATEAMS::statistics::Chain< ModelType >

Simulates the Markov chains specified by each Model.

Template Parameters
ModelTypeA Model type, like Bernoulli.

Example Usage

The below code draws 1,000 samples from plaquette Bernoulli percolation on \(\T^4_6\), the \(4\)-fold torus of scale \(6\).

using namespace ATEAMS;
using Model = models::Bernoulli;
using Complex = complexes::Cubical<Model::RingType>;
using Parameters = models::ModelParameters;
using Chain = statistics::Chain<Model>;
Complex plex({4,4,4,4});
Parameters params;
params.p = 0.5;
params.dimension = 2;
Model percolation(&plex, parameters);
Chain M(&percolation, 1000);
for (Chain::State state : M.simulate()) {

}
