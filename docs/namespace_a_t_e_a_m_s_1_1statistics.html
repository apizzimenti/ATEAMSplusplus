std::function< double(int)> ATEAMS::statistics::constant ( double C)
inline

Constant-temperature.

Parameters
CTemperature to return. If you want to see whether this temperature is valid (i.e. \(0 \leq 1-e^{-C} \leq 1\)), please set the DEBUG model parameter (e.g. in ATEAMS::models::SwendsenWangParameters) to true. If your provided temperature doesn't work, an assertion will fail.
Returns
Lambda function that always returns \(C\).

A convenience function for deciding when to stop invading (i.e. when to re-sample the spins over the included edges).

Parameters
ranksA vector of integers with values in \([1,\rank(H_d(X))]\), where \(\rank(H_d(X))\) is the rank of the \(d\)th homology group of \(X\).
Returns
A function that draws a value from ranks at uniform random, then subtracts \(1\).
