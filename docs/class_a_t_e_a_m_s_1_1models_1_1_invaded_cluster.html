Implements the plaquette invaded-cluster algorithm, which inserts satisfied cells (i.e. cells \(x\) such that \((\delta^{d-1}f_t)(x) = 0\)) in a uniform random order until some number of giant cycles are created, forming a subcomplex \(P_t\). Then samples \( f_{t+1} \sim \PK(- \mid P_t)\).
