Computes the persistent homology of a complex using \(\Z/2\Z\) coefficients.

Parameters
complexA (pointer to) a ATEAMS::complexes::Complex with a flat boundary matrix.
filtrationA vector that specifies the order in which to add the cells in the flat boundary matrix. For example, if we are doing 1-dimensional percolation in 2-d, then we switch up the ordering of the 1-d cells in the flat boundary matrix, but leave everything else.
dimensionThe percolation dimension.
Returns
A vector of percolation times.
